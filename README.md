# circum
 Camera-based Interactive Recognition and Control Using Mediapipe

![ab2](ab.gif)

只是一个简单的脚本，包含通过手势的鼠标控制模式和通过姿态与预设键位对键鼠的控制


鼠标控制：
✋抬起鼠标按键
✊按下左键
✌按下右键
👆左键点击（即迅速抬起放下）
🤟右键点击

预设键位为：
上键 手臂前伸
下键 手臂后拉（经常识别错）
左键 身体朝左
右键 身体朝右
鼠标左键 左手放下
鼠标右键 右手放下
